# Security Policy

## Supported Versions

We release periodic patches to protect our users from any security vulnerabilities. The following chart shows all eligible versions to recieve such patches:

| Version | Supported          |
| ------- | ------------------ |
| 1.0     | :white_check_mark: |

## Reporting Suspected Vulnerabilities

Please report any suspected security vulnerabilities to hello@post.in. You will receive a response from us within about 48 business hours. If the issue is confirmed, we will release a patch as soon as possible depending on complexity.
